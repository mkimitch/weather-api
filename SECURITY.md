# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch.

## Reporting a Vulnerability

If you believe you've found a security issue, please **do not open a public GitHub issue**.

Instead, report it privately:

- Email: **[mark.kimitch@gmail.com](mailto:mark.kimitch@gmail.com)**

Please include:

- A description of the issue and potential impact
- Steps to reproduce (proof-of-concept if possible)
- Any relevant logs, screenshots, or links

## Disclosure

I will acknowledge receipt and work on a fix as soon as possible. Once a fix is available, I'll coordinate a responsible public disclosure.
